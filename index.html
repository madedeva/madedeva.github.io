About Me

Halo, Saya I Made Deva Kerti Wijaya, Saya adalah seorang Web dan Mobile Developer, dan seorang Desainer Garfis. Saya telah mengerjakan beberapa project berupa aplikasi mobile dan desain. Saya menjadi Desainer Grafis di salah satu restoran korea yaitu Hansik by Ferbean.

Latar berlakang pendidikan saya yaitu Jurusan Rekayasa Perangkat Lunak di SMK Negeri 1 Negara, serta di perguruan tinggi sebagai mahasiswa Pendidikan Teknik Informatika, Universitas Pendidikan Ganesha. Saya juga mengikuti organisasi mahasiswa yaitu HMJ Teknik Informatika.
